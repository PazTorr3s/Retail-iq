## Retail Intelligence Dashboard

Dashboard de inteligencia comercial desarrollado con Next.js, que simula el funcionamiento real de un sistema de analítica empresarial utilizando APIs externas, datos procesados en el servidor y visualizaciones interactivas.

Este proyecto replica patrones reales de dashboards corporativos, manejo de datos, tipado estricto y arquitectura escalable.

## Tecnologías utilizadas

Next.js 14 (App Router)

TypeScript

Tailwind CSS

shadcn/ui

Recharts

API DummyJSON

## Instalacion

```bash
    git clone https://github.com/PazTorr3s/Retail-iq.git
    cd Retail-iq
```

## Instalar dependencias 
```bash
    npm install
```

## Ejecutar el proyecto de forma local
```bash
    npm run dev
```

## La aplicación se abrirá automáticamente en el navegador.
Si no ocurre, accede manualmente a:
```bash
    http://localhost:3000
```
